Apple Developer Resources

What Is It?

This is Apple's own library of all the software tools and documentation that one would need to create software on Apple's hardware. This includes resources for software development on OS X, iOS, watchOS, and now the newly-released tvOS.

What's so Special about It?

This is an official set of resources, straight from the hardware manufacturer, itself. This is where Xcode, Apple's Integrated Development Environment (IDE) can be found. Also, this is a great location for learning about Swift, a recent programming language presented by Apple.
