Networks

Definition

A network is a group of entities that can communicate, even though they are not all directly connected. Generally, a network consists of three or more such entities, and there must be a way to facilitate communication with entities that are not directly connected.

Producing a Network

To produce a network, there must be the following:

  1. A way to encode and interpret messages.
  2. A way to route messages.
  3. Rules for determining which entities get access to network resources

Latency

The time required for a message to travel from a given source to its destination. This is usually measured in seconds or milliseconds.

Bandwidth

The amount of information that can be transmitted in a given unit of time. Generally measured in bits per seconds.

Protocol

A rule set that determines how two entities communicate with each other. An example would be the Hypertext Transport Protocol that is used in the World Wide Web, with Get as an example of a message type.

Server

A computer (hardware) and/or computer program (software) that receives and fulfills requests made by other computers (clients). A server may operate within a local area network (LAN), a wide area network (WAN), or it could operate across the Internet.
