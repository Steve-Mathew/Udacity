Dictionaries

A dictionary is an advanced form of a list. Whereas a list contains elements - values, really - with a dictionary, those values are also named. As a result, a dictionary is a list that contains pairs of values; the name of the value, as well as the value, itself. When using string substitution, dictionaries can be incorporated easily, since the name of the values can be used, allowing the values to be changed, while leaving the rest of the code unchanged.

Web Template Engines

A Web Template Engine is a module to a given programming language. This module allows for more advanced HTML generation. Code can be executed within an HTML prior to its actual rendering. Examples of Web Template Engines are Django and Jinja.
